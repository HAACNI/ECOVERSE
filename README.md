# ECOVERSE
Environmental Education app
An interactive eco-awareness web project with animations and visuals
focused on sustainability and environmental responsibility.

Features
- Interactive eco-themed UI
- Animated falling leaves background
- 3D/visual assets for engagement
- Clean HTML, CSS, and JavaScript structure

Tech Stack
- HTML
- CSS
- JavaScript

Project Structure
index.html
style.css
main.js
assets (images & models)

How to Run
1. Clone the repository  
   `git clone <repo-link>`
2. Open `index.html` in a browser

Purpose
This project was built to explore creative web development
and interactive environmental storytelling.
